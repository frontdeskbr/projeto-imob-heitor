# Plataforma Imobiliária — Quickstart

## Rodar com FastAPI
```bash
docker compose --profile fastapi up -d --build
```

## Rodar com NestJS
```bash
docker compose --profile nestjs up -d --build
```

PgAdmin: http://localhost:8080  (crie uma conexão para host `db`, porta `5432`)

API: http://localhost:3000
